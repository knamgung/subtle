Subtle is an application where it detects the shots (e.g. Upper Body Shot, Face Shot, etc...) of the images the user uploads.

### Use Subtle

First clone or download Subtle

```
cd client
yarn install

cd ..

cd server
yarn install

yarn dev
```


### Stacks Used

#### Front-End
* Apollo
* ml5.js
* p5.js
* React.js
* tensorflow.js

#### Back-End
* express
* graphQL
* mongoose
* mongoDB
